# gdb调试命令



| 用途             | 语法                                                        |
| ---------------- | ----------------------------------------------------------- |
| 结构体格式化输出 | set print pretty on                                         |
| 打印变量类型     | ptype                                                       |
| 完整打印字符串   | set print elements 0                                        |
| 查看断点信息     | 1.`info breakpoints [list...]` <br>2.`info break [list...]` |
